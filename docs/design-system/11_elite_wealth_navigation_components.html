Component Specifications

Typography

Component Size Weight Line Height
Breadcrumb 12px Regular (400) 1.5
Tab (inactive) 14px Medium (500) 1.5
Tab (active) 16px SemiBold (600) 1.5

Spacing

  • Tab Padding: 12px vertical × 16px horizontal
  • Tab Active Padding: 10px vertical × 16px horizontal (border reduces visible space)
  • Breadcrumb Gap: 8px
  • Indicator Size: 24px
  • Border Width: 2px (active tab bottom border)

States

  • Default: Secondary color (#71A9CA)
  • Hover: Primary hover color (#4C8DB4)
  • Active: Primary color (#016991) + bottom border
  • Disabled: Disabled color (#BDBDBD) + opacity 0.6
