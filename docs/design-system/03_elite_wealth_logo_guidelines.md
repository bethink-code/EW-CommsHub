# Elite Wealth Logo Guidelines

**Version:** 1.0 | **Status:** Production Ready

---

## Logo Specifications

### Primary Logo
- **Format:** SVG (scalable)
- **Variants:** Full horizontal layout
- **Colors:** Two-color (Blue #016991 + Tangerine #EA8A2E)
- **Minimum Size:** 100px width
- **Usage:** Main branding, websites, documents

### Logo Mark (Icon)
- **Format:** SVG
- **Size:** Square, 1:1 aspect ratio
- **Colors:** Blue (#016991)
- **Minimum Size:** 32px
- **Usage:** App icons, favicons, social media avatars

### Stacked Logo
- **Format:** SVG
- **Layout:** Vertical stacking
- **Colors:** Two-color
- **Minimum Size:** 120px height
- **Usage:** Mobile, narrow spaces, social profiles

---

## Logo Variations

All logos available in:
1. **Full Color** (primary branding)
2. **Monochrome Blue** (professional contexts)
3. **White/Inverted** (dark backgrounds)
4. **Gradient** (decorative use)

---

## Clear Space & Sizing

### Minimum Clear Space
- Minimum 10px around logo in all directions
- Don't place other elements too close

### Sizing Rules
- Desktop: 200px–400px width
- Mobile: 120px–180px width
- Favicon: 32px–64px
- App Icon: 180px–1024px

---

## DO's and DON'Ts

**✅ DO:**
- Use approved color values
- Maintain aspect ratio
- Provide minimum clear space
- Use on brand-approved backgrounds

**❌ DON'T:**
- Distort or stretch the logo
- Change colors arbitrarily
- Add effects or filters
- Use on poor contrast backgrounds
- Reduce below minimum sizes
